<h1>Mario Kart Heardle</h1>
It's like Wordle, but with Mario Kart music!

Test your knowledge of Mario Kart Songs with this fun, wordle-style game. Play the song of the day, and see if you can guess the game and track it comes from in 6 guesses or less.
Too easy? Try changing the difficulty to hard mode to limit the playback time.<br>
